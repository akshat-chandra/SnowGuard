# SnowGuard: Data Quality Monitor

A Snowflake Native App that monitors data quality across your tables.

## Features
- Automated null percentage detection
- Row count trend analysis
- Schema drift detection
- Value distribution monitoring

## Getting Started
Once installed, open the SnowGuard dashboard from the Apps section in Snowsight.
The app will request access to ACCOUNT_USAGE views to inspect your table metadata.
